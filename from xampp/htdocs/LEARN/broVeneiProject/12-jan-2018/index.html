Collapse error THE FLOAT PROBLEM

when height have zero when inspect so the probelem is collapse because we use the float.

So to fix: we need to use before and after element.

We need to mention only the parent class not child class

        
            .navigation .second::before,
            .navigation .second::after{
                display: table;
                content: "";
            }
        
            .navigation::after{
                clear: both;
            }

so the collapse problem is fixed.

Or we can use the overflow property

            .navigation .second{
                overflow: auto;
            }

Then if there is some slidebar we can use the 

            .navigation .second{
                overflow: hidden;
            }

The last property of the style will overwrite the before property of the class.
